# Benchmark Data

A fixed snapshot of NASA/JPL small-body and planetary data. **Read-only** ‚Äî do
not modify, move, or delete anything in this directory. Load it at runtime with
`fetch` (relative paths); make no external network calls.

Provenance and exact query URLs are in [`provenance.json`](./provenance.json).
The snapshot is regenerated by `scripts/fetch-data.mjs` + `scripts/build-planets.mjs`.

## Files

| file | what | rows |
|---|---|---|
| `asteroids.json` | All known Near-Earth Objects (NEOs) with orbital + physical data | ~42,000 |
| `close-approaches.json` | NEO close-approach events to Earth | ~50,000 |
| `planets.json` | The eight planets' orbital elements + radii | 8 |
| `comets.json` | Comets, same element schema (optional overlay) | ~4,000 |
| `sentry.json` | CNEOS impact-risk summary (optional overlay) | ~2,200 |
| `provenance.json` | Sources, query URLs, retrieval timestamp | ‚Äî |

All three data files are **arrays of objects**. Values are numbers or `null`
(many physical fields are unknown for less-studied objects).

## Coordinate & element conventions

Orbital elements describe a heliocentric orbit in the **J2000 ecliptic** frame.
Angles are in **degrees**, distances in **astronomical units (au)**, time in
**days**. Each object carries its **own** `epoch` (a Julian date) ‚Äî propagate
each body from its own epoch.

Positions are **not** stored; they must be **computed** from the elements for a
chosen time (i.e. solve the orbit / Kepler's equation and advance the mean
anomaly by the mean motion). The Sun is the central body, at the origin, with a
mean radius of **696,000 km**.

### Orbital element fields (in `asteroids.json` and `planets.json`)

| field | meaning | unit |
|---|---|---|
| `a` | semi-major axis | au |
| `e` | eccentricity | ‚Äî |
| `i` | inclination | deg |
| `om` | longitude of ascending node (Œ©) | deg |
| `w` | argument of perihelion (œâ) | deg |
| `ma` | mean anomaly at epoch | deg |
| `epoch` | epoch of the elements | Julian date |
| `n` | mean motion (deg/day) ‚Äî present on planets; derivable elsewhere | deg/day |
| `per` | orbital period | days |
| `q`, `ad` | perihelion / aphelion distance | au |
| `tp` | time of perihelion passage | Julian date |

### `asteroids.json` ‚Äî additional fields

| field | meaning |
|---|---|
| `full_name`, `pdes`, `name` | display name, primary designation, IAU name (may be null) |
| `spkid` | unique JPL SPK-ID |
| `neo`, `pha` | booleans: Near-Earth Object / Potentially Hazardous Asteroid |
| `class` | orbit class code (e.g. `APO`, `ATE`, `AMO`, `IEO`) |
| `moid` | minimum orbit intersection distance with Earth (au) |
| `H` | absolute magnitude (brightness) |
| `G` | magnitude slope parameter |
| `diameter` | estimated diameter (km) ‚Äî often null |
| `albedo` | geometric albedo ‚Äî often null |
| `rot_per` | rotation period (hours) ‚Äî often null |
| `spec_B`, `spec_T` | spectral type (SMASSII / Tholen) ‚Äî often null |
| `first_obs` | date of first observation |

### `planets.json` ‚Äî additional fields

| field | meaning |
|---|---|
| `name` | planet name |
| `radius_km` | mean radius (km) |

### `comets.json` (optional overlay)

Comets, using the **same orbital-element fields** as asteroids (`a e i om w ma
epoch q per n tp`) plus `full_name`, `pdes`, `class`, `M1` (total magnitude), and
`diameter`. **Caution:** roughly half are hyperbolic or near-parabolic
(`e >= 1`) ‚Äî these have no period/mean-motion and must be propagated from `tp`
(time of perihelion passage), not from `ma`/`n`.

### `sentry.json` (optional overlay)

CNEOS Sentry impact-risk summary, one row per monitored object. Join to asteroids
on `des` ‚Üî asteroid `pdes`.

| field | meaning |
|---|---|
| `des`, `fullname` | designation / display name |
| `ip` | cumulative impact probability |
| `ps_cum`, `ps_max` | cumulative / max Palermo scale |
| `ts_max` | max Torino scale (0‚Äì10) |
| `range` | year range of potential impacts |
| `n_imp` | number of potential impacts |
| `diameter`, `h`, `v_inf` | size (km), abs. magnitude, encounter velocity (km/s) |
| `last_obs`, `last_obs_jd` | last observation date |

### `close-approaches.json`

One row per Earth close-approach event. Join to asteroids on
`des` ‚Üî asteroid `pdes`.

| field | meaning |
|---|---|
| `des` | object designation (matches asteroid `pdes`) |
| `cd` | close-approach date/time (calendar, TDB) |
| `jd` | close-approach time (Julian date) |
| `dist` | nominal approach distance (au) |
| `dist_min`, `dist_max` | 3œÉ distance bounds (au) |
| `v_rel`, `v_inf` | relative / unperturbed velocity (km/s) |
| `h` | absolute magnitude of the object |
